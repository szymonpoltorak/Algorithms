# What is this repo for ?
This repository is a place I store my algorithm implementations made during process of learning. There are sorting algorithms,
 data structures and maybe more.

# Technology Stack
* JDK 17.03 LTS,
* Apache Maven 3.8.3,
* Language Level Java 11,
* JUnit 5.8.2.
